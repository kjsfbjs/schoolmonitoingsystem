# School Management System (Flask + SQLite)

A **simple school management system** built with **Flask** and **SQLite**.  
It allows admins, teachers, and students to manage users, student records, upload marksheets, and import/export Excel files.

---

## **Features**

- **User Management**: Add, delete, and manage users with roles (admin, teacher, student).  
- **Student Management**: Add, update, delete student records.  
- **Marksheet Upload**: Upload PDF marksheets for students.  
- **Excel Import/Export**: Import student data from Excel files and export current data.  
- **Dashboard**: Role-based dashboard with total student count.  
- **Simple UI**: Built with HTML + Bootstrap for quick and clean interfaces.

---

## **Technologies Used**

- Python 3.x  
- Flask  
- SQLite (via SQLAlchemy)  
- Pandas (Excel handling)  
- Bootstrap 5
